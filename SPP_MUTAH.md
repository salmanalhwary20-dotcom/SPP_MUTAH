منصة University Projects | التحول الرقمي لإدارة المشاريع الجامعية
نظام متكامل لإدارة، تقييم، وتوثيق مشاريع التخرج والمسارات الأكاديمية.

🌐 نظرة عامة على النظام
منصة سحابية متطورة مصممة لخدمة الطلاب والمشرفين، توفر بيئة عمل تعاونية تضمن تنظيم المشاريع من الفكرة وحتى المناقشة النهائية. تم بناء النظام بهيكلية برمجية متينة تضم أكثر من 314 ملفاً ومجلداً لضمان أقصى درجات الكفاءة والأمان.

🚀 المميزات التقنية (Core Engine)
يعتمد النظام على بنية تحتية برمجية (Full-Stack) متكاملة:

نظام الحماية (Security): طبقات حماية متقدمة عبر LoginSecurity.php وتشفير البيانات advanced_encryption.php.

إدارة الفرق (Team Management): وحدة متكاملة لإدارة طلبات الانضمام، الدعوات، وصلاحيات الأعضاء (teams_manager.php).

التواصل الفوري: نظام دردشة متطور يدعم تبادل الملفات والصور بين أعضاء الفريق (chat_functions.php).

إشعارات ذكية: تنبيهات عبر البريد الإلكتروني والرسائل النصية القصيرة (PHPMailer & sms_service.php).

تعدد اللغات: دعم كامل للغتين العربية والإنجليزية (languages/ar.php).

📊 هيكلية المنصة (Project Architecture)
تم تنظيم المشروع بشكل احترافي لتسهيل الصيانة والتطوير:

لوحة تحكم الإدارة (Admin Dashboard): لمراقبة التقارير، تقييم الفرق، وإدارة المستخدمين.

مركز إدارة المشاريع: رفع الملفات، عرضها عبر file-viewer.php والمراجعة المستمرة.

المعرض الرقمي (Gallery): استعراض مخرجات المشاريع والصور والنماذج الأولية بشكل مرئي جذاب.

نظام الأرشفة والتقارير: توليد تقارير شاملة وإدارة سجلات الأخطاء (logs) لضمان استقرار النظام.

🛠 الأدوات والتقنيات المستخدمة
Backend: PHP (Object-Oriented Logic).

Database: MySQL (Relational Mapping).

Frontend: HTML5, CSS3 (Bootstrap), JavaScript.

Security: SHA-256 Encryption, .htaccess protection.

Services: SMTP (Email), SMS Gateway API.

سهولة الوصول: واجهة مستخدم سريعة الاستجابة تعمل على كافة الأجهزة.
